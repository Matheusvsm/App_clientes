## Instalar aplicação

Certifique-se de ter o Node instalado:
```
No diretorio Raiz:

npm install

Instalando Api:

cd api

npm install

```

## Rodar aplicação

Abra dois terminais para rodar Front e Api:
```
No diretorio Raiz:

npm start

Na Api:

cd api

node server.js

```

## Versões

```
- Node v18.16.1
- React v18.2.0
- Express v4.18.3

```
